
# SC TurkeyIPM

[![DOI](https://zenodo.org/badge/1245130722.svg)](https://doi.org/10.5281/zenodo.20318878)

Code for manuscript: Differences in wild turkey nest initiation and population growth rates between a hunted and nonhunted site in South Carolina, USA  

There are two scripts that can be used to view and graph the raw data and output. 
- 01_DataCleaning.R contains data cleanup and graphs of raw data
- 02_ModelFit.R contains the nimble model, loads the raw object needed for running, and outputs the model results
